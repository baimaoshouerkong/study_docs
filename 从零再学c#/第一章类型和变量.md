# 类型和变量
```
int age = 20; // 定义一个整数类型的变量age，并赋值为20

double pi = 3.14; // 定义一个双精度浮点数类型的变量pi，并赋值为3.14

float temperature = 36.5f; //   定义一个单精度浮点数类型的变量temperature，
                                并赋值为36.5,
                                注意要加上f后缀,因为默认的浮点数是double类型

decimal price = 19.99m; //  定义一个高精度小数类型的变量price，
                            并赋值为19.99,注意要加上m后缀,因为默认的小数是double类型。
                            因为计算机底层使用二进制表示小数
                            所以有些小数无法精确表示，
                            可能会出现精度问题，
                            比如1.0000变成1.00001。
                            decimal类型可以提供更高的精度，
                            适合用于金融计算等需要高精度的场景。

bool isStudent = true; // 定义一个布尔类型的变量isStudent，并赋值为true

char grade = 'A'; // 定义一个字符类型的变量grade，并赋值为'A',注意字符类型使用单引号括起来

string name = "Alice"; // 定义一个字符串类型的变量name，并赋值为"Alice"

Console.WriteLine("年龄: {0}", age); // 输出年龄: 20,占位符式
Console.WriteLine($"名字: {name}"); // 输出名字: Alice,字符串插值式
Console.WriteLine("成绩: " + grade); // 输出成绩: A,字符串连接式
```
